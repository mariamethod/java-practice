# java-practice
Сборник от учебни Java задачи, включващи основи на езика - ООП (наследяване, полиморфизъм, абстракция, капсулация), колекции, изключения и др. 
Проектите са организирани в отделни папки.

## Съдържание
shapes - задача за фигури - може да се въведат няколко фигури (правоъгълник, триъгълник, кръг). Програмата извежда площта на въведените фигури.

## Стартиране
Проектите са създадени в Eclipse. 
За да ги стартирате:
1. Отворете Eclipse
2. File / Import / Existing Projects into Workspace
3. Изберете съответната папка
